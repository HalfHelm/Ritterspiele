# Problem description
In den 2010ern gab es einen Zuwachs an Webseiten, auf denen NutzerInnen kurze Videospiele spielen können. Als diese aber in den Mainstream gelangten, nahmen Werbeinhalte stark zu, während die Qualität der Spiele abnahm. Heute gibt es immer noch einen großen Markt an potenziellen NutzerInnen, die Webseiten sind aber zu einem großen Teil kaum noch verwendbar.

# Solution description
Unser Ziel ist es, eine neue Webseite zu bauen, auf der NutzerInnen interaktive Inhalte nutzen und kurze Spiele spielen können, ohne von Pop-Ups und Werbeinhalten gestört zu werden.

# Technical environment
Wir werden voraussichtlich folgende Sprachen für unser Projekt verwenden: JavaScript, HTML und CSS. Für einen reibungslosen Ablauf werden wir ein Git-Repository erstellen und eine passende GUI verwenden. (zBsp. GitKraken)
